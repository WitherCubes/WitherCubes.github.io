Fire and Ice Mind Map

June 25, 2021 - By Sanjay Pavan



Fire and Ice
      |
      |
      |
      |
      |
      ----------------Poet----------------
      |                                   |
      |                                   |
      |                                   |
      |                                   ------- Robert Frost
      |
      |
      |
      |
      |
      -------------- Theme ---------------
      |                                  |
      |                                  |
      |                                  |
      |                                  ------- Only love, equality, mutual
      |                                          understanding and sympathy for one
      |                                          and all can help in establishing 
      |                                          peace on earth.
      |
      |
      |
      |
      |
      -- Poetic Devices -----
      |                     |
      |                     |
      |                     |
      |                     |
      |                     |
      |                     - Enjambment -
      |                                  |
      |                                  |
      |                                  |
      |                                  ------- From what I've tasted of desire I
      |                                          hold with those who favour fire.
      |                     |
      |                     |
      |                     |
      |                     - Alliteration -
      |                                    |
      |                                    |
      |                                    |
      |                                    ------- Favour Fire
      |                                    |
      |                                    |
      |                                    |
      |                                    ------- World Will
      |                      |
      |                      |
      |                      |
      |                      -- Anaphora --
      |                                   |
      |                                   |
      |                                   |
      |                                   ------- Some Say
      |                      |
      |                      |
      |                      |
      |                      --- Imagery --
      |                                   |
      |                                   |
      |                                   |
      |                                   ------- To say that for destruction,
      |                                           ice is also great.
      |
      |
      |
      |
      |
      --------------- Poem ---------------
                                         |
                                         |
                                         |
                                         ------- The poem first admits the fire
                                                 (greed, passion) to be the destroyer
                                                 of the world.
                                         |
                                         |
                                         |
                                         ------- But later admits that ice
                                                 (dislike, hatred) can also be the
                                                 destroyer of the world.
                                         |
                                         |
                                         |
                                         ------- Hence ice and fire, both are threats
                                                 to the world.
