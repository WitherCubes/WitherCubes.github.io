Dust Of Snow Mind Map

June 23, 2021 - By Sanjay Pavan



Dust Of Snow
     |
     |
     |
     |
     |
     ----------------Poet----------------
     |                                  |
     |                                  |
     |                                  |
     |                                  ------- Robert Frost
     |
     |
     |
     |
     |
     -------------- Theme ---------------
     |                                  |
     |                                  |
     |                                  |
     |                                  ------- The poet feels that small and
     |                                          unimportant changes can make
     |                                          great changes in our future.
     |
     |
     |
     |
     |
     -- Poetic Devices -----
     |                     |
     |                     |
     |                     |
     |                     -- Assonance -
     |                                  |
     |                                  |
     |                                  |
     |                                  ------- The prominance of vowels throughout
     |                                          the line (shook down on me).
     |                     |
     |                     |
     |                     |
     |                     --- Irony ----
     |                                  |
     |                                  |
     |                                  |
     |                                  ------- Rued
     |                                  |
     |                                  |
     |                                  |
     |                                  ------- Hemlock
     |                     |
     |                     |
     |                     |
     |                     - Enjambment -
     |                                  |
     |                                  |
     |                                  |
     |                                  ------- The way a crow shook down on me.
     |                     |
     |                     |
     |                     |
     |                     - Alliteration -
     |                                    |
     |                                    |
     |                                    |
     |                                    ------- Saved Some
     |                                    |
     |                                    |
     |                                    |
     |                                    ------- Has Heart
     |
     |
     |
     |
     |
     --------------- Poem ---------------
                                        |
                                        |
                                        |
                                        ------- The poet tries to convey that
                                                small unimportant changes could
                                                change our life.
                                        |
                                        |
                                        |
                                        ------- Those small changes are not only
                                                optimistic but inauspicious events
                                                too.
                                        |
                                        |
                                        |
                                        ------- The poet's mood was changed by
                                                just a crow shooking down snow from
                                                the hemlock tree.
