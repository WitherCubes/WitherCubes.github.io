Softwate I use

May 03, 2021

Operating System

I use Arch Linux as my operating system. I was once a Windows 10 user but then hated it after knowing how evil it is. I wish I could use Gentoo as my OS but I have a dumb laptop to compile and I love the AUR.

Programs

Web Browser

For my default web browser I use Brave. It is better than any proprietary alternatives like Google Chrome, Vivaldi, and Microsoft Edge because Brave doesn’t collect user data and sell it. Brave blocks unwanted ads, cookies, trackers, etc. by default. But I don’t use onlt Brave. I also use Hardened Firefox, Ungoogled Chromium and Tor for my private stuff. I am also testing keyboard driven browsers like Qutebrowser, Nyxt Browser, and surf during my free time.

Window Manager

I use both DWM because I like window managers a lot especially tiling window managers. I am next testing out Qtile and AwesomeWM.

Terminal Emulator

I use alacritty and st as my terminal emulator. Alacritty is fast, written in rust and GPU accelerated, st has minimal code and extensible through patching.

Shell

I use both zsh as my default shell because it is POSIX compliant and highly customiseable.

Text / Code editor

Currently I use Doom Emacs as my text editor because it is an amazing piece of software and vim when I am quickly editing a file in my terminal.

File Manager

Thunar is the file manager I use because it is easy for me to use a graphical file manager.

Documents, Mail

Thunderbird is my mail client, and I use Zathura to view pdf’s.

Video and audio

For both video and audio I use mpv because it is the best video player in my opinion.

Youtube Videos

There is great content in youtube to watch but the problem is that Google can’t stop spying on you and pushing you ads. So I use Ytfzf, a terminal youtube front-end to watch youtube videos.

Misc Software

  • Torrent: qBittorrent
  • Image Viewer: sxiv
  • Image Editor: GIMP
  • System Backups: Timeshift
