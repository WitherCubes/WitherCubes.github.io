Secure the Firefox Web Browser

May 11, 2021 3-minute read
software • mozilla • programs • browser

Most Firefox users think that firefox has security and privacy features out of the box. The answer is that it’s certainly better compared to other mainstream alternatives (like Microsoft Edge, Google Chrome).

Why do people trust that Firefox more in terms of security and privacy than Google Chrome? The answer is simple because Firefox is a Free and Open Source Software. This means that anyone can audit the code of firefox and submit bug fixes, exploits to improve firefox and inspect the code if any malicious code is there in the browser.

Proprietary Software like Google Chrome, Apple’s Safari, Microsoft Edge code are closed source, that means the code can’t be audited and in many scenarios proprietary software have trackers and malware to track the user because to collect the data.

Tracking protection

Let’s start by enabling tracking protection. Firefox can block trackers and maliciious scripts. There are three levels to tracking protection: Standard (the default), Strict and Custom. You can enable Strict protection but Firefox warns you that enabling this stronger setting may cause some websites to break.

Whether you enable Strict protection or stay with the Standard protection, I strongly advise setting the Do Not Track feature to ALWAYS. This will make firefox try to block trackers and may or may not work on all websites, but there’s no harm in enabling Do Not Track.

  • Go to Preferences
  • Privacy and Security
  • Enhanced Tracking Protection
  • Change from Standard to Strict, if you wish.
  • Scroll down to Do Not Track and set to ALWAYS!

Telemetry

There is telemetry built in to Firefox by default which collects user information to improve the browser’s experience and performance. Telemetry is a privacy concern, so let’s disable it.

  • Go to Preferences
  • Privacy and Security
  • Scroll down to Firefox Data Collection and Use
  • Uncheck them all (if you wish)

Changing search engine

Firefox uses Google as the default search engine because Mozilla is funded mostly by Google. Google is not at all a privacy respecting search engine. If privacy is your top priority, use DuckDuckGo as your search engine. It is privacy respecting and awesome.

  • Go to Preferences
  • Search
  • Default Search Engine
  • Change it to DuckDuckGo

Extensions

Do not install random extensions from the internet for your browser because who knows what it can do. Use uBlock Origin, it is and adblocker which is lightweight and easy on CPU and memory, Privacy Badger, which is an anti-tracking addon that was created by the EFF. By using this extension, you are helping the EFF in the fight for privacy and digital rights. Also use HTTPS Everywhere, which is an extension which forces websites to use https protocol even if you use a http link.
