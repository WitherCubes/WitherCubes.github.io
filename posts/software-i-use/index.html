Softwate I use

May 3, 2021 2-minute read
software • linux • programs

Operating System

I use Arch Linux as my operating system. I was once a Windows 10 user but then hated it after knowing how evil it is. I wish I could use Gentoo as my OS but I have a dumb laptop to compile and I love the AUR.

Programs

Web Browser

For my web browser I use Brave. It is better than any proprietary alternatives like Google Chrome, Vivaldi, and Microsoft Edge because Brave doesn’t collect user data and sell it. Brave blocks unwanted ads, cookies, trackers, etc. by default.

Desktop Environment / Window Manager

I use both DWM and Qtile window managers because I like window managers a lot especially tiling window managers.

Terminal Emulator

I use alacritty and st as my terminal emulator. Alacritty is fast, written in rust and GPU accelerated, st has minimal code and extensible through patching.

Shell

I use both Bash and Fish as my shell because why not.

Text / Code editor

Currently I use vim as my text editor and occasionaly use Visual Studio Code as my IDE with vim keybindings plugin installed. I am interested in learning to use Doom Emacs when I am free.

File Manager

PcManFM is the file managers I use because it is easy for me to use a graphical file manager.

Documents, Mail

Thunderbird is my mail client, and I use my browser to view pdf’s.

Video and audio

For both video and audio I use mpv because it is the best video player in my opinion.

Youtube Videos

There is great content in youtube to watch but the problem is that Google can’t stop spying on you and pushing you ads. So I use Freetube to watch youtube videos. I also use mpv to watch youtube videos.

Chat

Most of the chatting applications have a web client, so I use Ferdi, a messaging browser to access all my messages in a single application.

Misc Software

  • Torrent: qBittorrent
  • Image Viewer: sxiv
  • Image Editor: GIMP
  • System Backups: Timeshift
